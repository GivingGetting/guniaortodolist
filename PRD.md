# PRD — 待办清单应用

## 1. 产品概述

**产品名称**：待办清单（Todo List）
**定位**：轻量级个人任务管理工具，面向中文用户，运行于浏览器端，无需登录注册。
**核心价值**：快速记录任务、追踪完成进度、通过优先级和截止日期管理时间。

---

## 2. 目标用户

- 需要管理日常任务的个人用户
- 偏好简洁、无干扰界面的用户
- 不需要团队协作，数据存本地即可接受

---

## 3. 已实现功能（v1.0）

### 3.1 任务管理

| 功能 | 描述 |
|------|------|
| 新增任务 | 输入文本后按回车或点击 + 按钮添加 |
| 完成任务 | 点击圆形按钮切换完成/未完成状态，完成后文字划线变灰 |
| 编辑任务 | 悬停显示编辑按钮，弹窗内修改文本、日期、优先级 |
| 删除任务 | 悬停显示删除按钮，点击移除 |
| 拖拽排序 | 鼠标拖拽或键盘操作调整任务顺序 |

### 3.2 优先级

- 三档：**高**（红）/ **中**（橙）/ **低**（蓝）
- 新建时默认「中」优先级
- 任务左侧边框颜色 + 旗帜图标标识优先级

### 3.3 截止日期

- 日历选择器选择日期，不可选过去日期
- 截止状态智能提示：

| 状态 | 显示 |
|------|------|
| 已逾期 | 已逾期 N 天（红色） |
| 今天 | 今天到期（琥珀色） |
| 明天 | 明天到期（琥珀色） |
| 7 天内 | N 天后到期（主题色） |
| 7 天以上 | M月d日（灰色） |
| 已完成 | M月d日（灰色） |

### 3.4 筛选与统计

- 三种视图：全部 / 进行中 / 已完成
- 标签实时显示各分组任务数
- 底部统计：共 N 个任务，已完成 N 个

### 3.5 数据持久化

- 数据存储于 `localStorage`（key: `lovable-todos`）
- 刷新页面、关闭重开均保留数据

## 7. 技术约束

- 纯前端，无后端服务
- 数据持久化仅限 localStorage（容量约 5MB）
- 构建产物为静态文件，可部署至任意 CDN / 静态托管
